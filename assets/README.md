## Dicas

- Use formato `.jpg` ou `.webp` para fotos (melhor compressão).
- Otimize as imagens para web (ferramentas como TinyPNG ou Squoosh).
- Para os casos "Antes e Depois", mantenha o mesmo enquadramento e iluminação nas duas fotos.
- Fotos da equipe devem ter fundo neutro e boa iluminação.
- A imagem do hero deve ser de alta resolução e impactante.
